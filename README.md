PointGrey Java Interface
======================

A Java API for the PointGrey Flycapture 2 SDK

Note: the dll must be compiled and placed in a location java is aware of.
To compile the dll, the Flycapture sdk must be installed. http://ww2.ptgrey.com/sdk/flycap

This project is still in very early alpha stages but will be receiving regular updates and fixes. If you run into any trouble compiling or using this code, feel free to contact one of the developers at spacephlite@gmail.com or matt77561@gmail.com
